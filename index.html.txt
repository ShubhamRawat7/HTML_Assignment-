        <section id="html">
          <h1>What is HTML?</h1>
          <ul>
            <li>HTML stands for Hyper Text Markup Language</li>
            <li>HTML is the standard markup language for creating Web pages</li>
            <li>HTML describes the structure of a Web page</li>
            <li>HTML consists of a series of elements</li>
            <li>HTML elements tell the browser how to display the content</li>
            <li>
              HTML elements label pieces of content such as "this is a heading",
              "this is a paragraph", "this is a link", etc.
            </li>
          </ul>
        </section>
        <section id="html_element">
          <h1>What is an HTML element?</h1>
          <p>
            An HTML element is defined by a start tag, some content, and an end
            tag:
          </p>
          <p style="font-size: 20px">
            <span style="color: blue">&lt;</span>
            <span style="color: rgb(185, 0, 0)">tagname</span>
            <span style="color: blue">&gt;</span>Content goes here...
            <span style="color: blue">&lt;</span>
            <span style="color: rgb(185, 0, 0)">/tagname</span>
            <span style="color: blue">&gt;</span>
          </p>
          <span style="font-size: 14px"
            >The HTML element is everything from the start tag to the end
            tag:</span
          >
          <p style="font-size: 20px">
            <span style="color: blue">&lt;</span>
            <span style="color: rgb(185, 0, 0)">h1</span>
            <span style="color: blue">&gt;</span>My First Heading
            <span style="color: blue">&lt;</span>
            <span style="color: rgb(185, 0, 0)">/h1</span>
            <span style="color: blue">&gt;</span>
          </p>
          <p style="font-size: 20px">
            <span style="color: blue">&lt;</span>
            <span style="color: rgb(185, 0, 0)">p</span>
            <span style="color: blue">&gt;</span>My first paragraph.
            <span style="color: blue">&lt;</span>
            <span style="color: rgb(185, 0, 0)">/p</span>
            <span style="color: blue">&gt;</span>
          </p>
          <table class="table table-striped">
            <thead>
              <tr>
                <th scope="col">Start tag</th>

                <th scope="col">End tag</th>
